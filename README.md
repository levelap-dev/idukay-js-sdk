# idukay-js-sdk
Used to integrate apps with Idukay

# for Init
`npm install idukay-js-sdk`
## Set on js file.
### ES6

`import idukay from 'idukay-js-sdk';`

### ES5
`var idukay = `require('idukay-js-sdk');`

# for login
`idukay.login() //render pop up;`

# get user logged
`idukay.isUserLogged(); //return true if logged in.`
